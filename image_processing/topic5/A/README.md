## 考察

### 計算時間について

|手法|倍率[%]|実行時間[ms]|
|:--|--:|--:|
|最近傍法|33|0|
|最近傍法|270|31250|
|線形補間法|33|0|
|線形補間法|270|125000|

処理時間はtime.hを使用して取得したが1msかからないほど縮小は速いことが分かった．  
また，参照範囲が増えると処理が重くなることも分かった．

### クオリティについて

最近傍法では1Dot程度の細い線が消えている個所が，線形補間法では線が残っている．  
このことから参照範囲が増えると精度が上がることがわかる．
